# بوت تلجرام - الإصدار v4

## الجديد في هذا الإصدار
1. **أزرار متداخلة بلا حدود (زر داخل زر داخل زر...)**
   أي زر بتحطه بأي رد (`reply_key`) تقدر توصله بمفتاح جديد داخل `callback_responses`، وهذا المفتاح نفسه يقدر يملك أزراره الخاصة توصل لمفاتيح تانية، وهكذا لأي عمق تحبه. شوف مثال جاهز بـ 3 مستويات داخل `config.json`:
   `menu_info` ⬅️ `info_service` ⬅️ `info_service_details`.

2. **تحكم كامل بالوسائط: صور، فيديوهات، صوت، ورسائل صوتية**
   أي رد (رئيسي، رد زر، ترحيب، أو وداع) يقدر يحمل حقل `media`:
   ```json
   "media": { "type": "photo", "path": "media/example.jpg" }
   ```
   الأنواع المدعومة: `photo` (صورة)، `video` (فيديو)، `audio` (صوت/أغنية)، `voice` (رسالة صوتية)، `document` (ملف/مستند)، `animation` (GIF).
   ضع الملف داخل مجلد `media/` بجانب `config.json` واكتب مساره بحقل `path`.
   إذا ما بدك وسائط، خلي `media` تساوي `null` أو احذف الحقل بالكامل.

3. **توقيع تلقائي بآخر كل رسالة**
   بملف `config.json` تحت `settings`:
   ```json
   "settings": { "signature": "𝑺𝑬𝑫𝑫𝑰𝑮" }
   ```
   هذا النص رح ينضاف تلقائياً بآخر كل رسالة يرسلها البوت (رد رئيسي، رد زر، ترحيب، وداع...). خليه فاضي `""` إذا ما بدك توقيع.

## طريقة التشغيل
```bash
pip install -r requirements.txt
```
1. افتح `config.json` وحط توكن البوت بحقل `settings.token` (تجيبه من @BotFather بتلجرام).
2. شغّل البوت:
```bash
python bot_v4.py
```
3. (اختياري) شغّل المحرر البصري لتعديل كل شيء بدون كتابة كود:
```bash
python editor.py
```
افتح المتصفح على `http://localhost:8765` وعدّل الردود والأزرار والوسائط والتوقيع، ثم اضغط "حفظ الكل".

## إضافة زر يفتح قائمة فرعية
1. بأي رد، أضف زر بحقل `reply_key` باسم مفتاح جديد، مثلاً `"reply_key": "my_submenu"`.
2. داخل `callback_responses` أضف مفتاح بنفس الاسم:
```json
"my_submenu": {
  "reply": "هذا نص القائمة الفرعية",
  "media": null,
  "buttons": [
    [ { "text": "زر فرعي 1", "reply_key": "deeper_menu" } ],
    [ { "text": "⬅️ رجوع", "reply_key": "back_main" } ]
  ]
}
```
3. كرر نفس الخطوة لأي مستوى إضافي تحبه — ما في حد أقصى لعدد المستويات.

## إعادة تحميل التعديلات بدون إيقاف البوت
أرسل الأمر `/reload` داخل تلجرام (من نفس الجهاز اللي شغّل عليه البوت أو أي محادثة معه) بعد أي تعديل بـ `config.json`، وبيقرأ آخر نسخة فوراً بدون الحاجة لإعادة تشغيله.
